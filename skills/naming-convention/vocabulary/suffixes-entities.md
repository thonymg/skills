# Approved Entity Suffixes

Core business entities valid across all domains.
Add project-specific entities in `vocabulary/custom.md`.

---

## Identity & Access
| Suffix | Represents |
|:-------|:-----------|
| `User` | An authenticated person |
| `Session` | An active login context |
| `Token` | An auth or API credential |
| `Role` | An access control role |
| `Permission` | A single access right |

## Commerce
| Suffix | Represents |
|:-------|:-----------|
| `Order` | A purchase order |
| `Cart` | A shopping cart |
| `Item` | A line item inside a cart or order |
| `Product` | A sellable item |
| `Payment` | A financial transaction |
| `Invoice` | A billing document |
| `Subscription` | A recurring billing plan |

## Content & Communication
| Suffix | Represents |
|:-------|:-----------|
| `Message` | A chat or system message |
| `Notification` | An alert or push message |
| `Document` | A file or formal record |
| `Report` | A generated summary |

## Organisation
| Suffix | Represents |
|:-------|:-----------|
| `Team` | A group of users |
| `Project` | A scoped initiative |
| `Task` | A unit of work |
| `Event` | A calendar or system event |

## System
| Suffix | Represents |
|:-------|:-----------|
| `Config` | Application configuration |
| `Log` | An activity log entry |
| `Job` | A scheduled background process |
| `Request` | An inbound HTTP or API request |
| `Response` | An outbound HTTP or API response |
| `Error` | An error or exception |
| `Cache` | A temporary data store |
| `Query` | A DB or search query |

---

## Collection Suffixes
| Suffix | Represents |
|:-------|:-----------|
| `List` | An ordered collection |
| `Page` | A paginated subset |
| `Batch` | A group for bulk operations |
| `Results` | Output of a search or query |
