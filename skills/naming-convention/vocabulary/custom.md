# Custom Vocabulary

Add your project-specific words here.
All entries in this file carry the same authority as the core vocabulary files.
Any word listed here is approved for use in names.

---

## How to add an entry

Follow the same format as the core files.
Every entry must have:
- the word
- what it represents (one sentence)
- which file it extends (Prefix / Entity / Attribute / Infrastructure / UI)
